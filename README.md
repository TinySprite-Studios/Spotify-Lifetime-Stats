# Spotify Lifetime Stats

A simple local/browser tool for analysing Spotify Extended Streaming History JSON files.

## What it does

- Calculates lifetime listened hours
- Calculates lifetime listened days
- Shows equivalent years of music
- Counts total plays
- Counts unique artists and songs
- Shows top artists
- Shows top songs
- Shows hours per year
- Includes jump buttons for each stats section
- Exports a Markdown report
- Exports a yearly CSV

## Privacy

This tool processes files locally in your browser. Nothing is uploaded to a server.

## How to use

1. Request your Spotify Extended Streaming History from Spotify.
  - https://www.spotify.com/uk/account/privacy/
2. Extract the ZIP file.
3. Open `index.html`.
4. Select all files named like `Streaming_History_Audio_*.json`.
5. Click `Generate Stats`.
